Notes:

You can put many auto-validation checks on individual form elements with attributes, no scripting required.
  • Accept specifies accepted file upload types
  • List sets inputs to only accept <options> from a <datalist>
  • Min/Max for numbers
  • MinLength/MaxLength for any text
  • Multiple checks whether multiple emails or files are allowed.
  • Pattern. Yeah we got regex.
  • Required.
  • Step determines incremental values that are valid for numbers.
